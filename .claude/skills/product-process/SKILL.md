# Product Process (Generic)

## 0) Idea & Validation
- Define problem, target user, and success metric.
- Run a fast validation: landing page + waitlist or interviews.

## 1) Product & UX
- Draft user journey and key screens.
- Define MVP scope and avoid feature creep.

## 2) Data & Architecture
- Model core entities and relationships.
- Define API contracts and pagination/search.

## 3) Build
- DB schema + migrations.
- API routes + validation.
- UI components + pages.
- Analytics + SEO basics.

## 4) QA
- Smoke tests for core flows.
- API response checks.
- Performance sanity check.

## 5) Launch
- Configure env + deploy.
- Add sitemap/robots.
- Monitor errors and performance.

## 6) Growth & Ops
- Publish initial content.
- Set up analytics dashboards.
- Create feedback loop.

## 7) Iterate
- Review metrics weekly.
- Ship small improvements.
